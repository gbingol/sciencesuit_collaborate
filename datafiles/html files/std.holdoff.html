holdoff

Sets the hold state to off for the plot window
As of hold state being off, new plot windows will be displayed.

Syntax:

holdoff()

Use std.holdon(id) to initiate retaining the current plot.

Example:

>>id=std.plot(x,x^2)
>>std.holdon(id)
>>std.plot(x,x^3)
>>std.holdoff()

--following will be displayed on a new plot window
>>std.plot(x,std.cos(x))
