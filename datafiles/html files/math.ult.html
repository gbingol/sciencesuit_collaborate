ult

Returns a boolean, true if and only if integer m is below integer n when they are compared as unsigned integers.

Syntax:

ult(m, n) → boolean

Example Usage:

>>math.ult(2,3)
true

>>math.ult(-2,3)
false

>>math.ult(-2,math.maxinteger)
false

Note that the upper range of type unsigned integer is greater than the type signed integer.
