collectgarbage

An interface to the garbage collector and performs functions according to argument, opt.

  • "collect":performs a full garbage-collection cycle. This is the default option.
  • "stop":stops the garbage collector.
  • "restart": restarts the garbage collector.
  • "step":performs a garbage-collection step.
    The step "size" is controlled by arg (larger values mean more steps) in a non-specified way.
    If you want to control the step size you must experimentally tune the value of arg.
    Returns true if the step finished a collection cycle.
  • "setpause":sets arg as the new value for the pause of the collector. Returns the previous value for pause.
  • "setstepmul":sets arg as the new value for the step multiplier of the collector. Returns the previous value for step.

Syntax:

collectgarbage(opt [, arg])

 

Example:

>>print(collectgarbage("count"))
19.06640625

>>a={1,2,3,4}
>>print(collectgarbage("count"))
20.81640625

>>a=nil
>>print(collectgarbage("count"))
21.9296875

>>collectgarbage()
>>print(collectgarbage("count"))
19.294921875
