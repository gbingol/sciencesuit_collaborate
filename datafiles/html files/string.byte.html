byte

Returns the internal numeric codes of the characters s[i], s[i+1], ..., s[j].
The default value for i is 1; the default value for j is i.

Numeric codes are not necessarily portable across platforms.

Syntax:

byte (s [, i [, j]])

Example:

>>s="abcd"

>>string.byte(s,1)
97

>>string.byte(s,1,4)
97
98
99
100
