scatter

Plots scatter plot and also a library for different scatter plots.

Syntax:

  1. scatter(x=, y=, Name="")
  2. scatter.line(x=, y=, Name="")
  3. scatter.smooth(x=, y=, Name="")

returns the ID of the created plot window otherwise nil.

x, y: Vectors
Name: Name of the series ← string

 

Example:

>>x=std.sequence(-4, 4, 0.1)
>>y=std.dnorm(x)
>>std.scatter(x, y, "Standard Normal distribution")
