plot

Currently, an alias for the function std.scatter(x=, y=, Name="").

Syntax:

plot(x=, y=, Name="")

returns the ID of the created plot window.

x, y: Vectors
Name: Name of the series ← string

 

Example:

>>x=std.sequence(-4, 4, 0.1)
>>y=std.dnorm(x)
>>std.plot(x, y, "Standard Normal distribution")
