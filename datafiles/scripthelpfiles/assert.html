assert

Checks whether its first argument is not false and simply returns that argument
if the argument is false (that is, false or nil), assert raises an error.

Note:Such combination of if not ... then error end is so common that Lua has a built-in function called assert:

Syntax:

assert(v [, message])

 

Example:

>>a=6
>>assert(a==6,"Must be 6")
true     Must be 6

>>assert(a==7,"Must be 6")
stdin:1: Must be 6
