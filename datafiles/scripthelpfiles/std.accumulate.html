accumulate

Accumulates addable entries in an iteratable container.
Returns type entry and the number of entries.

Syntax:

accumulate(container=, x0=, func=nil)

 

container: an iteratable container,
x0: initial value
func: a unary function applied on each entry in the container

Examples:

>>t={1, 2, 3}
>>std.accumulate(t, 0)
6
3

>>v=std.util.tovector{1, 2, 3}
>>std.accumulate(v, 0, function(x) return x^2 end)
14 --sum of squares
3
