sub

Returns the substring of s that starts at i and continues until j; i and j can be negative.
If after some correction i is greater than j, the function returns the empty string.

Syntax:

sub (s, i [, j])

Example:

>>s="abcde"

>>string.sub(s,2)
bcde

>>string.sub(s,1,3)
abc

>>string.sub(s,-2)
de

>>string.sub(s,-3,-1)
cde
