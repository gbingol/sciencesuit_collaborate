tovector

Converts a given iteratable container to Vector.

Syntax:

tovector(container=)

returns either nil or a Vector and an integer.

Examples:

--3 numeric and 2 non-numeric
>>t={1, 2, "a", 3, "b"}

>>std.tovector(t)
1    2    3    COL
2

 

For example, for a matrix data type:

>>m=std.tomatrix{ {1, 2} , {3, 4} }
>>m
1    2
3    4

>>std.tovector(m)
1    2    3    4    COL
0
