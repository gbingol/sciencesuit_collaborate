qnorm

Computes the lower tail quantile (inverse of pnorm).
Returns a Vector or a number.

Syntax:

1) qnorm{p=,mean=0, sd=1}

2) qnorm(p=,mean=0, sd=1)

 

p: Cumulative probability (left-tailed) ← Vector / number
mean: Mean of the distribution ← number
sd: standard deviation ← number

Example Usage:

>>std.pnorm{q=0}
0.5

>>std.qnorm{p=0.5, mean=0, sd=1}
0
