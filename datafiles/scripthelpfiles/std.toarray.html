toarray

Converts a given iteratable container to Array.

Syntax:

toarray(container=)

returns Array, numeric and non-numeric entries.

 

From Lua Table

>>t={1, 2, "a", "b", 3}

>>std.util.toarray(t)
1     2    "a"     "b"     3
3
2

 

From Vector

>>v=std.rand(3)
>>v
0.0238247     0.656005     0.879824     COL

>>std.util.toarray(v)
0.0238247     0.656005     0.879824
3
0

 

From Matrix

>>m=std.rand(2, 2)
>>m
0.77254     0.39515
0.77577     0.24375

>>std.util.toarray(m)
0.77254     0.395152     0.775774     0.243749
4
0
