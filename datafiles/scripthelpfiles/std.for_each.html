for_each

Applies func to each entry in an iteratable container.
If successful returns the index of the entry, otherwise nil.

Syntax:

for_each(container=, func=, ...) )

container: An iteratable container
func: A unary, binary or multivariable function
... 2nd, 3rd... arguments of the parameter func.

Example:

>>t={1, 2, 3, 4}
>>std.for_each(t, function(x) return x^2 end)
1    4   9   16

>>t={1, 2, 3, 4} >>y=2 >>std.for_each(t, function(x, y) return math.pow(x, y) end, y) 1    4   9   16
