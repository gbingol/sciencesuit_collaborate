holdon

Retains current plot when adding new plots on a selected plot window.

Syntax:

holdon(WindowID)

where WindowID is the ID of the plot window.

Use std.holdoff() when retaining current plot is no longer needed.

Example:

>>id=std.plot(x,x^2)
>>std.holdon(id)
>>std.plot(x,x^3)
>>std.holdoff()
