polyval

Evaluates a polynomial (anxn + an-1xn-1 +..+ a0) at a given data or dataset.

Syntax:

polyval(v=, arg=) → number / Vector

where argument v must be of type Vector. If arg is a single data point the function returns a number;
however, if arg is a vector of data points then function returns a vector.

 

Examples:

x y
0 4
1 10
2 26
3 58
4 112
5 194

 

>>coef=std.polyfit(x, y, 3)

>>std.polyval(coef, 3.5)
81.875

>>v=std.tovector{3.5, 4.5}
>>std.polyval(coef, v)
81.875    149.125    COL
