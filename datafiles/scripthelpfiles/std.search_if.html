search_if

Finds the entries in an iteratable container satisfying a given condition.
Returns the entries and the number (integer) of entries satisfying the condition.

Syntax:

search_if(container, function)

Arguments:

container: An iteratable container
func: A Unary Predicate function

 

Example:

>>t={10, 20, 30, 40}

>>std.search_if(t, function(x) return x>15 end)
20    30    40
3

>>std.search_if(t, function(x) return x<15 end)
10
1
