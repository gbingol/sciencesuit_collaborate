find

Finds the location of the first occurence of an entry in an iteratable container.
If successful returns the index of the entry, otherwise nil.

Syntax:

find(container=, value=, tol=1E-5)

 

container: An iteratable container
value: Value being searched
tol: Tolerance, only used when searched for numeric entries. ← number

Example:

>>t={1, 5, 3, 8, 8, -1}
>>std.find(t,8)
4

>>m2=std.util.tomatrix{ {1, 2} , {4, 5} , {7, 8} }
>>std.find(m2,5)
4
