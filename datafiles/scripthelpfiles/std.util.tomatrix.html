tomatrix

Converts a given suitable iteratable container to Matrix.

  1. tomatrix(vector=, nrows=, ncols=)
  2. tomatrix(table=, nrows=, ncols=)
  3. tomatrix(table=)
  4. tomatrix(range=, nrows=, ncols=)

Please note that Matrix can only contain numeric values.

 

From Vector

>>v=std.util.tovector{1, 2, 3, 4, 5, 6}

>>m=std.util.tomatrix(v, 2, 3)
>>m
1     2     3
4     5     6

 

From 1-D Lua table

Please note that table must contain only numeric entries.

>>t={1, 2, 3, 4, 5, 6}

>>m=std.util.tomatrix(t, 2, 3)
>>m
1     2     3
4     5     6

 

From 2-D Lua table

Please note that table must contain only numeric entries.

>>t={ {1, 2}, {3, 4}, {5, 6} }

>>m=std.util.tomatrix(t)
>>m
1     2
3     4
5     6
